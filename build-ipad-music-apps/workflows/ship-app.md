# Workflow: Ship iPad Music App

## Required Reading
1. references/appstore-release.md
2. references/background-audio.md
3. references/audio-session-routing.md

## Process
1. Confirm bundle id, versioning, and signing profiles.
2. Validate background audio and route changes.
3. Build Release archive via CLI.
4. Run final smoke tests on device.
5. Upload and submit to App Store Connect.

## Success Criteria
- Release archive built and validated
- App passes device smoke test
- Submission created in App Store Connect
