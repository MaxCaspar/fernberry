# Workflow: Debug Workflow

## Required Reading
1. `references/testing-debugging.md`
2. `references/expressions.md`

## Process
1. Reproduce the issue using pin data.
2. Inspect execution history and node outputs to locate the failing node.
3. Verify expressions, data shapes, and node configuration.
4. Apply the minimal fix and re-execute the workflow.
5. Confirm the error is resolved and outputs are correct.

## Success Criteria
- Error resolved and workflow executes successfully.
- Outputs are correct after re-execution.
