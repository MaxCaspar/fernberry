# Workflow: Optimize Workflow

## Required Reading
1. `references/performance.md`

## Process
1. Identify the slowest or most memory-intensive nodes via execution history.
2. Apply batch processing, memory optimizations, or node-level changes.
3. Re-execute the workflow with pin data and compare performance.
4. Document changes and any tradeoffs.

## Success Criteria
- Execution time or memory usage improved measurably.
