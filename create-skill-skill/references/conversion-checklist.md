# Conversion Checklist

1. Identify input type (skill/prompt, agent spec, command).
2. Extract name, description, and core instructions.
3. Choose Codex skill name (lowercase, hyphens).
4. Draft SKILL.md frontmatter and Objective.
5. Map instructions to Essential Principles and workflows.
6. Add references for reusable knowledge.
7. Verify routing and file paths.
8. Add optional command hook if requested.
9. Provide a dry-run example and assumptions.
