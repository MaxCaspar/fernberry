# Using Templates

Templates are reusable output structures that Codex copies and fills in.

## When To Use

- Reports, specs, or plans with consistent structure
- Customer-facing or compliance-sensitive docs

## How To Use

- Store templates in `templates/`.
- In workflows, specify when to use them and how to fill them.

## Example

```
templates/plan.md
```

```markdown
# Plan

## Goals
- ...

## Steps
1. ...
```
