# Workflow: Audit a Skill

## Required Reading
1. references/skill-structure.md
2. references/core-principles.md
3. references/common-patterns.md

## Process

1. Check frontmatter
   - name matches directory
   - description explains what and when

2. Check structure
   - headings are clear
   - SKILL.md is concise
   - workflows and references are linked

3. Check execution quality
   - steps are actionable
   - success criteria are measurable

4. Report gaps and fixes

## Success Criteria

- Issues and fixes documented
- No missing references or workflows
- Structure and clarity improved
