# Workflow: Upgrade to Router Pattern

## Required Reading
1. references/recommended-structure.md

## Process

1. Identify distinct intents in the current skill.
2. Create `workflows/` and split procedures by intent.
3. Move reusable knowledge into `references/`.
4. Replace SKILL.md body with intake and routing sections.
5. Add indexes for workflows and references.

## Success Criteria

- Separate workflow files exist
- SKILL.md routes correctly
- References are linked and used
