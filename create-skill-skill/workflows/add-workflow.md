# Workflow: Add a Workflow

## Required Reading
1. references/recommended-structure.md

## Process

1. Locate the skill directory.
2. Create or update `workflows/{workflow-name}.md`.
3. Add sections:
   - Required Reading
   - Process
   - Success Criteria
4. Update SKILL.md to list the new workflow.

## Success Criteria

- Workflow file exists
- SKILL.md references the workflow
- Steps and success criteria are clear
