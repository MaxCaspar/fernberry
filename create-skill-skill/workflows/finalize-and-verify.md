# Workflow: Finalize And Verify

## Required Reading
1. references/conversion-checklist.md

## Process
1. Validate structure
   - Confirm SKILL.md frontmatter and directory name match.
   - Ensure workflows and references exist and are linked.

2. Quick dry-run
   - Provide a minimal example input and intended output summary.
   - Ensure the steps are actionable without missing dependencies.

3. Document outputs
   - List the files created.
   - Note any assumptions made during conversion.

## Success Criteria
- Structure is valid and complete.
- A dry-run walkthrough is documented.
- Outputs and assumptions are clearly stated.
