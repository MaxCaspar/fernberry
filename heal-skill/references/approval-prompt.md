# Reference: Approval Prompt

## Purpose
Explicit approval prompt before making edits.

## Details
```
Should I apply these changes?

1. Yes, apply and commit all changes
2. Apply but don't commit (let me review first)
3. Revise the changes (I'll provide feedback)
4. Cancel (don't make changes)

Choose (1-4):
```

Wait for the user response. Do not proceed without approval.
